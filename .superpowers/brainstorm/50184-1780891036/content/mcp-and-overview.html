其他体验优化

1. 工具错误处理增强

当前工具错误返回 JSON {"error": "..."}，LLM 需自行解读。统一为人类可读提示 + 操作建议。

  • K线不存在 → "请先在看盘页下载该标的日K数据"
  • 回测无结果 → "请先在回测页完成一次回测，或告诉我策略名我来触发"
  • 选股条件不适用 → "当前仅支持涨幅/换手率/均线金叉条件"

2. 对话上下文窗口管理

长对话 token 超限问题。

  • ChatStore 加 message_count 上限（如 50 条）
  • build_api_messages 取最近 N 轮对话
  • 工具调用结果超过 2000 字符自动截断

3. 工具调用进度提示

回测/选股可能耗时，LLM 不应无响应。

  • 工具调用 > 3s 时 UI 显示 "正在执行 xxx..."
  • LlmSignals 新增 tool_call_started / tool_call_finished 信号
