# tdx-stock-diagnose

单票综合诊断 Skill，数据全部来自**通达信问小达 MCP**（`mcp_tdx_tdx_wenda_quotes`），不读本地 K 线。

## 适用场景

- "帮我诊断一下这只票"
- "技术面 + 基本面怎么样"
- "MACD / 资金流 / PE 多少"
- "券商怎么看、有什么研报"

## 推荐调用方式

### 方式 A：一键诊断工具（优先）

```
diagnose_stock(symbol="002230.SZSE")
```

终端会自动并行查询问小达多个维度（行情、技术指标、财务、资金流、研报），返回结构化 JSON。

### 方式 B：直接调用 MCP（灵活追问）

工具名：`mcp_tdx_tdx_wenda_quotes`

参数：
- `question`：自然语言查询，格式 `[股票名称/代码]+[查询内容]`
- `range`：默认 `AG`（A 股）

单票诊断常用 question 模板：

| 维度 | question 示例 |
|------|---------------|
| 行情 | `科大讯飞002230最新行情涨跌幅成交量` |
| 技术指标 | `科大讯飞002230MACD KDJ RSI技术指标` |
| 财务估值 | `科大讯飞002230市盈率ROE财务指标` |
| 资金流 | `科大讯飞002230主力资金流向` |
| 研报 | `科大讯飞002230最新研报评级` |

**注意**：问小达每次调用仅支持**单个品种**；多维度分析请分多次调用后综合，或直接用 `diagnose_stock`。

## 输出解读指引

回答用户时按以下结构组织（均须基于工具返回，禁止编造）：

1. **行情快照**：现价、涨跌幅、所属行业
2. **技术面**：MACD / KDJ / RSI 等指标读数与含义（描述性，非买卖建议）
3. **基本面**：PE、ROE 等估值与盈利指标
4. **资金面**：主力净流入/流出
5. **研报**（若有）：机构观点摘要，注明来源与日期

## 研报数据说明

- 当前通达信 MCP **仅暴露** `tdx_wenda_quotes`（问小达），**无**独立研报/F10 工具
- 问小达对「最新研报评级」类问题常只返回行情 + 外部 `show_url`，**不含**券商研报正文
- `reports` 为空时：如实告知无数据；可说明已尝试 Tushare 降级（若已配置 `research_report` 权限）
- **禁止**引导用户在 zak 内打开 F10（zak 无此页面）

## 合规

- 禁止给出具体买入价、卖出价、仓位建议
- 禁止将指标读数包装为确定性预测
- 数据缺失时如实说明，不要假设本地 K 线
