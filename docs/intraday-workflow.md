# 盘中工作流（短线为主）

> **阶段**：框架期。把 [交易体系](./trading-system.md) 七大模块串成 **盘前 → 盘中 → 盘后** 可执行路径。  
> **默认体验说明**：产品以极致短线为**研究主路径**，但信号区默认 Profile 为 `short_swing`（`AshareShortBreakoutStrategy` 短线放量）；首次打开自选页可选择其他风格。布局预设默认「盘中」。

---

## 1. 风格与页面映射

| 交易层级 | 周期 | 主用页面 | 主用策略/工具 |
|----------|------|----------|---------------|
| 极致短线 | 1–3 日 | 雷达、市场、自选持仓 | 龙头选股、LimitBoard 信号、OvernightExit overlay、情绪周期 |
| 短线波段 | 3–10 日 | 雷达展望、自选信号 | `AshareShortBreakoutStrategy` |
| 中线观察 | 2–8 周 | 回测、团队分析、信号 | `AshareDoubleMaStrategy` / `AshareTrendMaStrategy` |

详见 [策略配置方案](./strategy-profiles.md)。

## 2. 总流程

```text
┌─ 盘前（9:00 前 / 前夜）────────────────────────────────────────────┐
│ ① 情绪预期：昨涨停结构、连板高度、成交额                          │
│ ② 次日计划：3–5 只观察股 + 总仓位 + 允许模式（TradingPlan）       │
└────────────────────────────┬──────────────────────────────────────┘
                             ▼
┌─ 盘中（9:30–15:00）──────────────────────────────────────────────┐
│ ③ 择时闸：emotion_cycle 顶栏 → 做不做、几成                       │
│ ④ 选股：雷达龙头 / 连板梯队 → 自选池 + 信号区（≤10）              │
│ ⑤ 执行辅助：信号区 + 分 K + 登记持仓（记账，非下单）               │
│ ⑥ 风控：单笔 2% 计算器、日内 −5% 提示                              │
└────────────────────────────┬──────────────────────────────────────┘
                             ▼
┌─ 盘后（15:00 后）────────────────────────────────────────────────┐
│ ⑦ 复盘：笔记流水 + Playbook + CSV 导出                           │
│ ⑧ 迭代：调整信号区、更新次日计划、导出龙头 CSV                     │
└───────────────────────────────────────────────────────────────────┘
```

## 3. 逐步说明

### 3.1 盘前

| 步骤 | 操作 | 页面/工具 | 状态 |
|------|------|-----------|------|
| ① 环境 | 看昨涨跌停、连板、板块主线 | [市场页](./market-page.md)、[雷达页](./radar-page.md)（盘后刷新） | **已有** |
| ② 计划 | 写观察名单、仓位上限、买卖条件 | 交易计划对话框 + 同步自选 | **已有** |
| 数据 | 计划/信号区标的下载日 K | 本地页 / 选股结果「下载日 K」 | **已有** |

**AI**：「帮我做次日计划」→ `propose_trading_plan`（**已有**）。

### 3.2 盘中

| 步骤 | 操作 | 页面/工具 | 状态 |
|------|------|-----------|------|
| ③ 择时 | 确认阶段与建议仓位 | [情绪周期](./emotion-cycle.md) 顶栏芯片 | **已有** |
| ④ 选股 | 主线龙一、共振确认 | [雷达选龙头](./radar-leader-screening.md) | **已有** |
| ④b | 退潮则空仓 | emotion gate | **已有** |
| ⑤ 监控 | 自选 → 信号区（≤10） | 自选信号区 | **已有** |
| ⑤b | 登记买入 | 持仓区 + T+1 提示 | **已有** |
| ⑥ 风控 | 浮亏 −5% 异动 | 持仓异动 + 交易参数 | **已有** |
| ⑥b 飞书 | 选股完成 / 阶段变更 | [消息通知](./notifications.md) | **已有** |

**池子限额**：

| 池 | 上限 | 用途 |
|----|------|------|
| 交易计划 | 3–5 | 盘前次日候选 |
| 信号区 | 10 | 当日监控执行 |
| 自选池 | 50 | 总跟踪 |
| 持仓记账 | `POSITION_MAX_ITEMS` | 在途头寸 |

### 3.3 盘后

| 步骤 | 操作 | 工具 | 状态 |
|------|------|------|------|
| ⑦ 复盘 | 每笔：理由、模式、盈亏 | 笔记流水 + Playbook | **已有** |
| ⑧ 导出 | 龙头池 / 选股 run CSV | 选股 Hub 导出 | **已有** |
| ⑧b 飞书 | 选股完成 / 退潮 | [消息通知](./notifications.md) | **已有** |
| ⑧ AI | 今日龙头结构 + 明日观察 | 雷达「盘后解读」/ `build_eod_leader_prompt` | **已有** |

## 4. 按情绪阶段的日路径

| 阶段 | 盘中重点 | 少做 |
|------|----------|------|
| 冰点 | 空仓或 1 成试错；只看不动 | 打板、追高 |
| 启动 | 雷达首板 + 二板；计划内 3 只 + 信号区 | 重仓龙头 |
| 高潮 | 龙头选股 + 共振；信号区监控 | 杂毛补涨 |
| 分歧 | 龙一低吸；减仓 | 跟风、后排 |
| 退潮 | **不新开**；持仓隔日卖规则 | 一切买入 |
