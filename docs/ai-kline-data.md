# AI 功能与本地 K 线

各功能对**本地日 K** 的依赖与下载建议。路由见 [AI 数据路由](./ai-data-routing.md)。

---

## 结论

**不要求全市场下载日 K。** 优先保证：自选池 + 常分析标的 + 回测标的。

| 数据源 | 用途 |
|--------|------|
| Redis | 实时价、条件选股 preset |
| 问小达 MCP | 综合诊断、形态全市场扫描 |
| Tushare | 财务、估值、部分因子 |
| 本地日 K | 技术面、策略信号、回测 |

---

## 依赖等级

| 功能 | K 线 | 无 K 线时 |
|------|------|-----------|
| 当前行情 | 无 | Redis 快照 |
| `get_bars_*` / `technical_snapshot` / 策略信号 | **强** | 提示下载 |
| 历史走势 / 情景分析 | 优先本地 | 不足 → MCP |
| 综合诊断 | 无 | MCP |
| 团队分析 risk/strategy | 部分 | 不足标 N/A |
| 条件选股 / Recipe | 无 | Redis + Tushare |
| 形态选股 | 降级 | MCP 优先，失败扫本地（≤1200） |
| 回测解读 | **强** | 须先回测有数据 |

---

## 下载建议

| 优先级 | 范围 |
|--------|------|
| P0 | 自选池全部 + 当前选中标的 |
| P1 | 回测常用标的 |
| P2 | 形态离线扫描备用 |

入口：自选「下载日 K」、数据管理页、`cli.py data download-batch`。
