# 自选页说明

`Ctrl+2`。投研**记账 + 规则辅助**，非券商下单。Profile 见 [策略配置方案](./strategy-profiles.md)。

---

## 1. 池子与分组

| 池 | 上限 | 用途 |
|----|------|------|
| 自选池 | 50 | 雷达/选股「加自选」写入此处 |
| 信号区 | 10 | 当日策略监控 |
| 持仓记账 | 20 | 在途头寸 |
| 交易计划 | 3–5 | 盘前候选（[交易计划](./trading-plan-journal.md)） |

**自选** Tab 管全量排序；自定义分组（≤10）仅筛选视图，一只可属多组。雷达/选股只写自选池，右键 **加入分组**。

---

## 2. 布局与预设

```text
分组 Tab → 上下文条 → 主表/多维 → 信号区 → 持仓区
右侧：图表 / 五档 / 笔记
```

工具栏 **盘中 / 登记 / 复盘** 控制区段折叠；**信号 n · 持仓 n** 一键显隐；点 **持仓 n/20** 进入持仓专注。

---

## 4. 信号区

≤10 只监控名单；按 Profile 展示买/卖/观望。主表多选加入；交易时段默认 30 分钟巡检；**问 AI** 用 `build_signals_ai_prompt`。默认 `AshareShortBreakoutStrategy`（`short_swing`）。

---

## 5. 持仓区

手工登记成本/数量/买入日；浮盈、T+1、退出信号、隔日规则、异动角标。添加时计划外/退潮 toast；header 含 Profile、建议 vs 实际仓位、待卖/T+1/浮亏筛选。**问 AI** / `evaluate_overnight_exit`（`ultra_short` 有隔日规则列）。异动可推飞书（默认关）。持仓 `notes` 与 [笔记中心](./stock-notes.md) 不合并。

---

## 6. 多维与其它

**表格 | 多维** 切换卡片网格（2/3/4 列）；单击联动图表。右侧 **StockNotePanel**；`Ctrl+L` 悬浮球；工具栏批量回测、下载日 K。

典型路径：`雷达/选股 → 自选 →（可选）信号区 → 登记持仓 → 复盘`。见 [盘中工作流](./intraday-workflow.md)。
